# Todoアプリ

シンプルなデスクトップ用 Todo 管理アプリです。  
タスクの追加・削除・一覧表示ができます。

---

## 動作環境

- OS：Windows
- Python 3.x（標準ライブラリのみ使用・追加インストール不要）

---

## 起動方法

`run_app.bat` をダブルクリックして起動してください。

---

## 使い方

### タスクを追加する

1. 画面上部の入力フィールドにタスク名を入力します
2. **追加** ボタンをクリック、または **Enter キー** を押します
3. 入力したタスクが一覧に追加されます

> ※ 何も入力せずに追加しようとすると、警告メッセージが表示されます

### タスクを削除する

1. 一覧から削除したいタスクをクリックして選択します
2. **削除** ボタンをクリックします
3. 選択したタスクが一覧から削除されます

> ※ タスクを選択せずに削除しようとすると、警告メッセージが表示されます

---

## データの保存について

- タスクはアプリと同じフォルダ内の `tasks.json` に自動で保存されます
- アプリを閉じても、次回起動時にタスクが引き継がれます
- 追加・削除のたびに即時保存されます

---

## 画面構成

```
┌─────────────────────────────────────────┐
│  [タスクを入力...]          [追加]       │
├─────────────────────────────────────────┤
│  タスク1                                │
│  タスク2                                │
│  タスク3                                │
│  ...                                    │
├─────────────────────────────────────────┤
│                              [削除]     │
└─────────────────────────────────────────┘
```

| 要素 | 説明 |
|------|------|
| 入力フィールド | タスク名を入力するテキストボックス |
| 追加ボタン | 入力したタスクをリストに追加する |
| タスク一覧 | 登録済みタスクの一覧（スクロール対応） |
| 削除ボタン | 選択中のタスクをリストから削除する |
